Bare-Metal

hypathia is aimed at organisations that cannot or don't want to utilise public cloud providers like AWS or GCE.

Self-Contained

None of the features or components require external commercial services or functions. This does not mean that external services cannot be access, but the cluster will work without them.

Self-Service

User are able to use all features without the need to open support tickets or talk to an admin.

Automation

The design of hypathia is focused on automating all operations of the cluster, including updating and provisioning.

Shared Resources

hypathia allows the efficient deployment of workloads, from the very small to the giant in a resource efficient manner by sharing computing resources among the users.

License

Most of hypathia is licensed under the Apache 2.0 license. Some components may require a differing license.
